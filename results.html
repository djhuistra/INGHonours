THE RESULTS

The Code

The result of the project is 11 different code bases. Each code base has reached up to four progress points. Below you can find an overview of all code bases and direct links to each of their progress points. The four different progress points:
  • PP 1: The base system
  • PP 2: The protocol extension
  • PP 3: The first extension set
  • PP 4: The second extension set
In addition, some code bases started with the same foundation, after which teams split up and followed different extension routes. Each codebase is therefore marked with a route. More information about the two routes can be found on the Setup page. The developers also wrote reports after each progression point to reflect on their approach. The report bundle contains all reports related to a certain repository.
